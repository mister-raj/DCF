# HUL DCF Valuation Model

This repository contains a **Discounted Cash Flow (DCF) valuation model for Hindustan Unilever Limited (HUL)** built to estimate intrinsic equity value and compare it with the current market price.

---

## What This Model Does

* Forecasts **5-year operating performance** (Revenue, EBITDA, FCF)
* Estimates **Enterprise Value and Equity Value** using DCF
* Compares **Implied Share Price vs Current Market Price**
* Highlights **Upside / Downside** based on intrinsic valuation
* Includes **sensitivity to key assumptions** (WACC, terminal growth)

---

## Key Outputs

* **Current Stock Price:** ₹2,354
* **Implied Share Price (DCF):** ₹657
* **Upside / (Downside):** **–72%**

> Interpretation: The market is pricing in significantly higher growth and/or lower risk than assumed in the base-case DCF.

---

## Core Assumptions

* **Revenue growth:** Conservative, based on historical trends
* **Margins:** Stable to modest expansion
* **WACC:** India large-cap consumer benchmark
* **Terminal growth:** Long-term sustainable growth assumption
* **Capex & Working Capital:** Linked to revenue

(All assumptions are explicitly modeled and adjustable.)
